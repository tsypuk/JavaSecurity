var.log_root    = "/usr/local/var/log/lighttpd"
var.server_root = "/usr/local/var/www"

Store evil2.dtd and evil3.dtd in /usr/local/var/www directory


To run lighttpd:

#### lighttpd -f /usr/local/etc/lighttpd/lighttpd.conf
